## Print ⬜️ cube using C++

**Code 1**
```
#include<iostream>
using namespace std;

int main(){
    int n;
    cout<<"Enter the rows for Cube: ";
    cin>>n;
    for (int i=0;i<n;i++){
        for (int j=0;j<n;j++){
            cout<<"* ";
        }
        cout<<endl;
    }
    return 0;
}
``` 
**Output**
```
Enter the rows for Cube: 4

* * * *
* * * *
* * * *
* * * *
```
